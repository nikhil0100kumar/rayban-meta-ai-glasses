A premium, "vibe-coded" 3D immersive website for the Ray-Ban Meta AI smart glasses. This project features a high-performance scroll-driven experience, combining cinematic visuals with a modern, glassmorphic UI.

---

## 🛠️ Created With
This project was developed using a cutting-edge AI-assisted workflow:
- **Nano Banana Pro**: For sophisticated code generation and component architecture.
- **Veo 3.1**: For visual inspiration and design consistency.
- **Antigravity (Google DeepMind)**: For orchestrating the build, refining the animation logic, and final deployment.

---

## 🚀 Tech Stack
- **Frontend Core**: HTML5, Modern CSS3 (Variables, Grids, Advanced Transitions).
- **Animation Engine**: 
  - **GSAP (GreenSock)**: Orchestrating complex, choreographed scroll animations.
  - **ScrollTrigger**: Binding UI transitions directly to the scroll progress.
- **Smooth Interaction**: 
  - **Lenis Smooth Scroll**: For a premium, fluid scrolling experience across all devices.
- **Visual Performance**: 
  - **HTML5 Canvas**: Powers a 201-frame high-performance WebP sequence (`frames/`), creating an indistinguishable 3D effect without the overhead of a WebGL engine.
- **Typography**: 
  - **DM Sans** (via Google Fonts): A clean, geometric sans-serif for a premium tech aesthetic.
- **Utilities**: 
  - **FFmpeg**: Used in the asset pipeline to extract and optimize 4K video frames into efficient WebP sequences.

---

## ✨ Features
1. **Immersive Canvas Sequence**: A frame-by-frame 3D rotation of the glasses that stays perfectly in sync with the user's scroll.
2. **Glassmorphic UI Panels**: Premium "frosted glass" containers with real-time blur and depth effects.
3. **Circular Reveal Wipe**: A cinematic sweep transition between the hero view and the interactive 3D container.
4. **Interactive Stats Counter**: Animated hardware specifications that count up as they enter the viewport.
5. **Ambient Gradient Orbs**: Dynamically positioned, soft-glowing background elements reflecting the Meta AI brand identity.

---

## 📂 Project Structure
- `index.html`: The core structure and library integrations.
- `css/style.css`: The "vibe-coded" premium styling.
- `js/app.js`: The "Engine" — handles preloading, GSAP choreography, and the Canvas renderer.
- `frames/`: Optimized 201-frame asset sequence.
- `extract_frames.js`: Script used to process raw media into optimized frames.

---

## 🏃 Local Setup
1. Clone the repository.
2. Ensure you have the `frames/` directory populated (generated by `extract_frames.js`).
3. Open `index.html` in any modern web browser.
4. For the smoothest experience, use a local server to avoid CORS issues with frame preloading.
